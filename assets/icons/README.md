# Tab Bar Icons

This directory contains placeholder icons for the bottom tab navigation.

## Required Icons

### Rounds Icon (Grid)

- `rounds-outline.png` (24x24)
- `rounds-outline@2x.png` (48x48)
- `rounds-outline@3x.png` (72x72)
- Design: 4 small squares arranged in a 2x2 grid

### Settings Icon (Gear)

- `settings-outline.png` (24x24)
- `settings-outline@2x.png` (48x48)
- `settings-outline@3x.png` (72x72)
- Design: Simple gear outline

## Current Status

The current PNG files are minimal placeholders. They should be replaced with proper icon designs.

## Generating Icons

You can use any icon design tool (Figma, Sketch, Illustrator) or online icon generators to create these icons. Make sure they:

- Are outlined/stroke style (not filled)
- Have transparent backgrounds
- Match the specified dimensions
- Use a simple, clean design suitable for tab bar navigation

## Alternative: Using SVG

If you prefer to use SVG icons, you can create SVG files and convert them to PNG using tools like:

- Online converters (e.g., CloudConvert, Convertio)
- Command line tools (ImageMagick, Inkscape)
- Design software export features
